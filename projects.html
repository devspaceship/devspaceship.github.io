Live Happiness Map - coming soon

Using the Twitter API, we estimate a 'happiness' score per country using tweets from all around the world. To calculate this score, we use sentiment analysis methods and try to understand how they works. Once all the countries have an estimate, we visualize this data by coloring a world map in function of this score.
